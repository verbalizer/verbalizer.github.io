The verbalizer is a tool that generates natural language explanations in English for derivations in ontologies. The current prototype is restricted to an EL fragment of description logics. However, the tool is continuously being extended and improved. This software is based on Java and available under the Apache 2.0 license.

You find some support on how to use the verbalizer at the documentation section. Go to Publications to read up some applications or techniques regarding the verbalizer. In the links section you can find out more about other projects, websites or tools related with the plugin. Have a look at the tool in the screenshots. Download an example ontology for testing here.
