On this page you find information about how to use the verbalizer plugin.

  • Tutorials

To get the plugin up and running in Protege 5 you must at first download the archive. Then unzip the file and copy or move the jar file to the plugins folder of your Protege installation (under Protege 5, this is default Java/plugins). Start Protege and open/classify an ontology (e.g. this small example ontology). Navigate to an inferred axiom and click the questionmark. In the poped up window you can choose Proofed Based Explanation.
