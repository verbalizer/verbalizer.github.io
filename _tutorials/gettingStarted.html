--- layout : tutorial title : Getting started with Verbalizer ---

This small tutorial will guide you through your first few steps with the verbalizer plugin. We will load an ontology and get an explanation for an inferred subsumption.

  1. Firstly, open Protege.
  2. Load an existing ontology or generate a new one. You can download a small ontology here.
  3. Start a reasoner (FaCT++is recommended).
  4. Switch to the entities tab. If the plugin was installed correct, you should see small question mark icons on the right site in the class description view. If you don't see this view, enable it via Views > Class views > Description
  5. Click the question mark icon of the inferred subsumption that you want to get the explanation from. A window pops up.
  6. Now from the drop down menu, choose Verbalized Proofbased Explanation.
  7. You will notice, that some expressions are colored. If you hover over them, a tooltip text does appear.
